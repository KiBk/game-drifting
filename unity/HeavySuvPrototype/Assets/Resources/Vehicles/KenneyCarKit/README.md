# Kenney Car Kit subset

This directory contains `hatchback-sports.fbx` and its `colormap.png` palette
from Kenney's **Car Kit 3.1**.

- Source: https://kenney.nl/assets/car-kit
- License: Creative Commons Zero (CC0 1.0)
- Original license text: `License.txt`

The game uses the imported body as an optional cosmetic model. Its bundled
wheel meshes are hidden so the existing WheelCollider-driven wheels remain the
authoritative animated wheels for both body choices.
